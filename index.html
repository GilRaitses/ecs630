Behavioral Simulation of Drosophila Larvae

Data engineering, DOE design, and event-hazard modeling for stimulus-driven behavioral simulation. This project develops a stimulus-response modeling framework using event-hazard methods from survival analysis to predict behavioral events (turns, stops, reversals) as functions of LED stimulus intensity, temporal history, and contextual features.
